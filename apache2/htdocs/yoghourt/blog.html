Intro

By Jeremie

When I first started web design, I learned a lot from free templates taken from OSWD - Open Source Web Design and other similar websites.

The Template Yoghourt is, for me, a good way to give back to the Open Source Community. I hope you will make good use of it and learn as much as I could learn from others.

I also want to thank Pat Heard for his advices and his support when I first started web design. His help has been more than welcome.

History

By Jeremie

Version 2.0 - Sept 18th, 2007:

This will be the last version of this template. I have improved it as much as I could and I will now devote my time to the Wordpress version of this theme.

In this new version, the background always sticks in the middle of the page (even when you resize to a small window), the footer has a back-to-top button, each post contains a calendar and the menu and copyright text is smaller...

Version 1.3 - Sept 12th, 2007:

In this new version, I have added a footer and changed few things. I hope you like the result.

Version 1.2 - Sept 5th, 2007:

This new template is the result of many comments and suggestions from other web designers on private Google groups. The main criticisms were "text too big" and "not enough spacing". I have corrected it in this new version. Also the main content is organized as posts to give an example of how it would look on a blog. I will soon adapt it to a Wordpress template so come back to visit my website in the next few weeks.
